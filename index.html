Tools the I used to create the game

Unreal Engine 4 is a set of integrated tools designing and building games, simulations and visualizations for game developers. For the game, I used the Paper 2D Tile Sets and Tile Maps. But this software is not limited to that, developers can also create 3D games.

GIMP is a cross-platform image editor available for GNU/Linux, OS X, Windows and more operating systems. It is free software, you can change its source code and distribute your changes. I used this software to create all the characters before I uploaded them to Unreal Engine 4 to create the game. It is best practice to set the file in the correct format before starting creating the characters. For my game, it is necessary to have standard tile with the width 64px and tile height 64 px. If I want a bigger character I can multiple the number with this size. For example, if I want a character with a double height compared to my normal character so it would be the width 64px and tile height 128px. The image should be in PNG and the resolution would be 150ppi and transparent background.

Not to mention, there is a tool that is life saver for me when I design my character which is Adobe color themes. It helps me choosing harmonious and appealing color combination for my design game.

Target Demographics

The target audience for the game would be kids. The age range can be 3 years old and above.

Personality should be playful, lovely and somebody who loves Christmas and Winter decorations. The game attracts new players because I create all the designs, and this game attracts people who love the cold winter atmosphere, but also enjoy the warm atmosphere inside.

Not only that, it also brings the Halloween 's fear and adventure spirit, because the game begins at the beginning of Halloween. Players want to play this game if they love adventure and want to be a hero for all children to save Christmas and wake up Santa Claus.

Levels

There are three main levels in this game. Each level has one main evil that the main character needed to defeat. Each evil has its especial damage ability. First level, this is where the main character gets out of his home and start to find the gifts with a cold dark night. Here Peter must climb up the mountain on the dirty rock and dirty dust road. At this level, Peter must face with the rock evil who uses rock to damage the main character.

Second level is where Perter goes through a forest on his way to the mountain. Here he must go through a tunnel and faces the fire evil which uses fire to damage and kill Peter.

The third one is where he meets a winter storm and there are snow everywhere. Here Peter is finally got to the mountain and faces his last fight with the snow evil. Once he defeats the evil and can collect the gifts and saves Christmas of all Children in the world. Cheer for Peter!
